DevsDen  [Blog Web Applciation for Developers] with admin dashboard control.
Tech Stack: MERN 

Frontend: 
-React.js (Typescript)
-Tailwind CSS
-Shadcn UI

Backend:
-Node/Express.js (Typescript)

Database:
-MongoDB
-Firebase

Tools:
-JWT Authentication
-Google Authentication
-Redux Toolkit
-Axios
